# Example Artifacts

This directory contains example artifacts that demonstrate the structure and content expected in each phase of the
3-pass plan workflow.

## Files

- `spec-example.yaml` - Example of a spec artifact generated in the Spec (Plan) pass
- `planned-diffs-example.md` - Example of planned diffs documentation generated in the Spec (Plan) pass
- `scaffold-example.yaml` - Example of a scaffold artifact generated in the Scaffold (Implement) pass
- `scaffold-report-example.md` - Example of a scaffold report generated in the Scaffold (Implement) pass
- `stabilize-example.yaml` - Example of a stabilize artifact generated in the Stabilize (Validate) pass
- `stabilize-report-example.md` - Example of a stabilize report generated in the Stabilize (Validate) pass
- `task-example.yaml` - Example of a task file that would be processed by the 3-pass plan workflow

## Usage

These examples can be used as references when implementing the 3-pass plan workflow. They follow the schemas defined in
`.ai/schemas/` and demonstrate the expected structure and content for each type of artifact.
