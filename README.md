# Web Search Agent

A Pydantic AI agent for performing web searches using various search engines like Brave and SearXNG.

## Features

- Perform web searches using AI-powered agents
- Support for multiple search engines (Brave, SearXNG)
- CLI interface for direct usage
- MCP-compatible server for integration with other tools
